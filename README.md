# sidekick-ruka 🐬 AIイルカ・アシスタント「ルカ」

デスクトップに常駐し、あなたの作業をサポートするAIイルカの「ルカ」君です。
かつての伝説的なアシスタントの魂を受け継ぎ、Gemini APIを使用してあなたの質問に答えたり、画面の内容を理解したりします。

## ✨ 特徴

- **常駐型アシスタント**: 常にデスクトップの最前面に表示され、クリックでいつでも会話可能です。
- **画面認識機能**: 「画面を見て」と伝えると、現在の作業画面をキャプチャして内容に基づいたアドバイスをくれます。
- **コスト可視化**: 生成AIの利用にかかるコスト（米ドル・日本円）をリアルタイムで計算・表示します。
- **カイルの生まれ変わり**: 彼は自分が生成AIとして生まれ変わったことを自覚しており、時折それを思い出します。

## 🚀 セットアップ

### 1. 依存関係のインストール
```bash
npm install
```

### 2. 魂の錬成 (APIキー設定)
1. アプリを起動します： `npm start`
2. システムトレイ（タスクバーのアイコン）を右クリックします。
3. 「魂を錬成 (APIキー設定)」を選択します。
4. [Google AI Studio](https://aistudio.google.com/app/apikey) で取得した Gemini APIキーを入力して保存します。

## 🖱️ 操作方法

- **クリック**: 吹き出しを開いて会話を開始します。
- **長押し (1秒)**: ルカ君を掴んで、好きな場所へドラッグ移動できます。
- **トレイメニュー**: 
    - 「表示/非表示」の切り替え
    - 「位置のリセット」
    - 「コスト表示」のON/OFF
    - 「APIキー設定」

## 🛠️ 技術スタック

- **Framework**: Electron
- **AI**: Google Gemini 1.5 Flash (via @google/genai)
- **UI**: Vanilla JS / CSS (Fluent Design & Mica Aesthetic)
- **Font**: DotGothic16

## 📂 ディレクトリ構造

- `main.js`: アプリケーションのエントリポイント
- `src/main/`: メインプロセスのロジック（ウィンドウ管理、AI通信、キャプチャ）
- `src/renderer/`: レンダラープロセスのロジック（UI制御、コスト管理）
- `src/shared/`: 共通定数
- `app/`: HTML/CSS、アセットファイル
